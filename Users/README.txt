

1. Before proceeding please open up models/db-settings.php

2. Create a database on your server / web hosting package.

3. Fill out the connection details in db-settings.php

4. PTU_CSE_SYLLABUS_GUIDE supports MySQLi and requires MySQL server version 4.1.3 or newer.

5. To use the installer visit http://yourdomain.com/install/ in your browser. PTU_CSE_SYLLABUS_GUIDE will attempt to build the database for you.
